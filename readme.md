# How to use

- Clone repo
- Run `npm install`
- Run `npm run deploy`
- Run `npm run watch`
- Login with one of these users (you can leave the password field empty)
    - marsUser
    - keplerUser
    - proximaUser
    - saturnUser
    - uranusUser
    - venusUser
- Enjoy
